# 并查集
## [题目链接](https://www.acwing.com/problem/content/838/)

## 标签
模板题，并查集

## 时空复杂度

**时间复杂度** $O(\log n)$

**空间复杂度** $O(n + m)$

## 代码
```cpp
#include <iostream>
using namespace std;
const int N = 100010;
int n,m;
int p[N];
int find (int x) {
    if (x == p[x]) return x;
    return p[x] = find (p[x]);
}
int main () {
    cin >> n >> m;
    for (int i = 1;i <= n;i++) p[i] = i;
    while (m--) {
        char ch;
        int x,y;
        cin >> ch;
        cin >> x >> y;
        if (ch == 'M') p[find (x)] = find (y);
        else cout << (find (x) == find (y) ? "Yes" : "No") << endl;
    }
    return 0;
}
```
