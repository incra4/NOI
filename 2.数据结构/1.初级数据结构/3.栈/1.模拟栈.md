# 手写栈
## [题目链接](https://www.acwing.com/problem/content/830/)

## 标签
模板题，栈

## 代码
```cpp
#include <iostream>
using namespace std;
const int N = 100010;
int n;
int stk[N],top = -1;
int main () {
    cin >> n;
    while (n--) {
        int x;
        string s;
        cin >> s;
        if (s == "push") {
            cin >> x;
            stk[++top] = x;
        }
        else if (s == "pop") top--;
        else if (s == "query") cout << stk[top] << endl;
        else cout << (top == -1 ? "YES" : "NO") << endl;
    }
    return 0;
}
```
